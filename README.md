\# Workout Tracker API



Це RESTful API для відстеження спортивних вправ та робочої ваги, розроблене за допомогою \*\*ASP.NET Core Web API (C#)\*\*.



\## 🚀 Як запустити проєкт локально



1\. Склонуйте репозиторій:

&#x20;  ```bash

&#x20;  git clone https://github.com/cah9gan/WorkoutTrackerAPI.git

2\. Перейдіть у папку проєкту:

cd WorkoutApi

3\. Запустіть проєкт за допомогою .NET CLI:

dotnet run

4\. API буде доступне за адресою https://localhost:<порт>/api/exercises. Ви також можете відкрити /swagger, щоб переглянути автозгенеровану документацію UI.

\## 📝 Використання API



У документації Postman додані приклади відповідей (Examples) для кожного статусу:

\- \*\*200 OK\*\* — при успішному отриманні списку.

\- \*\*201 Created\*\* — при створенні нової вправи.

\- \*\*204 No Content\*\* — після успішного оновлення (PATCH) або видалення (DELETE).

\- \*\*404 Not Found\*\* — якщо вправи з таким ID не існує.
